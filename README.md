# Retro Flash & HTML5 Game Portal

A simple static website that lets you play HTML5 games and classic Flash games using the self‑hosted Ruffle emulator. Designed for GitHub Pages, with a clean folder structure and easy game management.

## Features
- Play HTML5 games directly in the browser
- Load Flash `.swf` files using the Ruffle emulator (no plugins needed)
- Simple `flash-game.html` loader using URL parameters
- Organized folders for games, assets, and emulator files
- Fully static — works on GitHub Pages with HTTPS
